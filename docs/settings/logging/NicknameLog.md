## Syntax  
`-settings nicknamelog <off|channel>`

## Aliases  
`nicknamelogs, nicklog, nicklogs`

## Options  
`#channel` - Sets the output channel for the logs  
`off` - Disables the filter

## Examples  
`-settings nicknamelog #logs`
